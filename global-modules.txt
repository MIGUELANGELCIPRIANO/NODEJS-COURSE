Es posible instalar una dependencia de manera global en nuestro computador para trabajar con ella en múltiples proyectos mediante el comando 'npm i -g', como por ejemplo:

npm i -g nodemon

En este caso, se instalaría nodemon como una dependencia global en nuestro computador y no debería instalarse como dependencia interna dentro de nuestro proyecto.
Solo bastaría con ejecutar el comando directamente en nuestra consola. 
Sin embargo, hay que considerar que si dicho proyecto se sube a un repositorio remoto, sería mejor aclarar, en la documentación para cualquier otro usuario,
la necesidad de instalar la dependencia antes de ejecutar el proyecto. 
O para evitar este inconveniente, simplemente instalarla de manera interna dentro del proyecto como cualquier otra dependencia.
